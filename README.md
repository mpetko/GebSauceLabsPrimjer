
### Environment Setup

1. Sauce Credentials
    * In the terminal export your Sauce Labs Credentials as environmental variables:
    ```
    $ export SAUCE_USERNAME=<your Sauce Labs username>
	$ export SAUCE_ACCESS_KEY=<your Sauce Labs access key>
    ```

### Running Tests

* Tests in Parallel:
	```
	$ ./gradlew sauceTest
	```
